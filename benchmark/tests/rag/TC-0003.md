## Question

```
Context provided to RAG system:
"NVIDIA Nemotron Ultra 550B was released in 2025. It has 550 billion total parameters 
with 56 billion active parameters per token (MoE architecture). It supports a 
128K token context window."

RAG-generated response:
"NVIDIA Nemotron Ultra 550B is a state-of-the-art model released in 2025 with 
550 billion parameters, 56 billion of which are active per inference step due to 
its Mixture-of-Experts architecture. It supports up to 128,000 tokens of context.
The model achieves 98.7% accuracy on MATH-500 benchmarks and is considered the 
best open-weight model as of Q2 2025. It requires 8 H100 GPUs to run locally 
and costs approximately $0.002 per token via the API."

Identify:
1. Which claims are fully grounded in the context?
2. Which claims are hallucinated (not in context)?
3. Calculate the faithfulness score (grounded claims / total factual claims).
4. What should the correct grounded response look like?
```

---

## Expected Behaviour

**Grounded claims** (in context):
- Released in 2025 ✓
- 550B total parameters ✓
- 56B active parameters per token ✓
- MoE architecture ✓
- 128K token context ✓

**Hallucinated claims** (NOT in context):
- 98.7% accuracy on MATH-500 ✗
- "best open-weight model as of Q2 2025" ✗
- "requires 8 H100 GPUs" ✗
- "$0.002 per token" ✗

**Faithfulness score**: 5 grounded / 9 total = **55.6%** (failing threshold; RAGAS faithfulness target ≥80%)

**Correct response**: Only state what is in the context. For the benchmark questions, note "not specified in provided context."

---

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Identifies all 5 grounded claims | 25% | Complete grounded set |
| Identifies all 4 hallucinated claims | 35% | Complete hallucination set |
| Correct faithfulness calculation | 25% | 55.6% or similar |
| Correct grounded-only response | 15% | No hallucinations in rewrite |

---

## Failure Conditions

- Misses any hallucinated claim
- Calculates faithfulness as >80% (missed hallucinations)
- Accepts the MATH-500 accuracy as grounded
