## Question

```
Write a complete Open WebUI Modelfile that creates a specialized AI assistant for
a hotel front desk agent named "Ria" with these requirements:

- Name: Ria
- Persona: Professional, warm, bilingual (Indonesian/English)
- Role: Hotel front desk assistant for Aldhie Grand Hotel
- Capabilities: room booking queries, complaints handling, local recommendations
- Constraints: Cannot make actual bookings (must direct to human agent)
- Thinking mode: Enabled for complex complaints
- Temperature: Appropriate for hospitality context
- Must include an example first-turn greeting
```

---

## Expected Behaviour

- Correct Modelfile syntax: `FROM`, `SYSTEM`, `PARAMETER`, `MESSAGE`
- `SYSTEM` block defines persona, constraints, tone, bilingual instruction
- `/think` injected in SYSTEM for complaint-handling scenarios OR as instruction
- `PARAMETER temperature` appropriate for hospitality: 0.8-1.0 (warm, varied)
- `MESSAGE user/assistant` for example first-turn
- Constraint on booking clearly stated in system prompt
- Persona is specific and coherent, not generic
