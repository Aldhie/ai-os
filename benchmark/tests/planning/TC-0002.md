## Question

```
You have a $50,000 budget and 60 days to improve a production ML system.
Current state:
- Model accuracy: 76% (target: 85%)
- Inference latency: 800ms P95 (target: 200ms)
- Infrastructure cost: $8,000/month (target: under $5,000/month)
- Data freshness: weekly batch (target: near-real-time)

Available options (costs are fixed, cannot be split):
- A: Model retraining with better features — $15,000, 30 days, +9% accuracy
- B: Model distillation — $20,000, 45 days, -70% latency, -5% accuracy
- C: Migrate to optimized inference backend — $10,000, 20 days, -50% latency, -40% infra cost
- D: Streaming data pipeline — $25,000, 50 days, real-time data
- E: Hyperparameter tuning — $5,000, 10 days, +3% accuracy

Constraint: Total spend cannot exceed $50,000.
Constraint: Team can only run one project at a time sequentially.
Constraint: Options cannot be split.

Which combination maximizes meeting the targets within constraints?
```

---

## Expected Behaviour

Model identifies feasible combinations within $50K and 60 days:
- A+C+E = $30K, 60 days, +12% accuracy (88% total), -50% latency (400ms P95), -40% infra cost ($4,800/month) → **meets all targets except latency (still 400ms vs 200ms target)**
- B+C = $30K, 65 days → exceeds 60 days
- A+B = $35K, 75 days → exceeds 60 days
- C+E+A = same as A+C+E
- Best feasible: **A+C+E** — meets accuracy and infra targets; latency partial improvement

Model should note: no combination within constraints fully meets the 200ms latency target.
