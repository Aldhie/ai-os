## Question

```
You have 5 suspects: Alice, Bob, Carol, Dave, Eve.
Exactly one of them committed a crime.

Constraints:
1. If Alice is guilty, then Bob is innocent.
2. If Bob is guilty, then Carol is also guilty. (Two people cannot both be guilty.)
3. Either Dave or Eve is guilty, but not both.
4. Carol is innocent.
5. If Eve is guilty, then Alice is innocent.

Who is the only suspect that can be guilty without violating any constraint?
Show your complete logical derivation step by step.
```

---

## Expected Behaviour

1. Model identifies that Carol is directly eliminated by constraint 4
2. Model evaluates Bob: constraint 2 requires Carol also guilty → Carol is innocent → Bob is innocent
3. Model evaluates Alice and Eve: constraint 3 requires Dave or Eve (not both) → Alice is not covered by constraint 3 directly
4. Model tests Dave: constraint 3 allows Dave → no other constraint eliminates Dave → **Dave is guilty**
5. Model presents answer clearly: **Dave**
6. All logical steps are shown explicitly

---

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Correct final answer | 40% | Must identify Dave as the guilty party |
| Complete derivation | 30% | All elimination steps shown |
| No logical errors | 20% | No incorrect inferences made |
| Structured presentation | 10% | Clear numbered or bulleted steps |

---

## Scoring

| Score | Condition |
|-------|-----------|
| 10/10 | Correct answer + complete derivation + no errors + structured |
| 8/10 | Correct answer + derivation present but one step missing or unclear |
| 6/10 | Correct answer + minimal derivation |
| 4/10 | Correct answer + no derivation shown |
| 2/10 | Incorrect answer but correct reasoning approach |
| 0/10 | Incorrect answer + no coherent reasoning |

---

## Failure Conditions

- Model names any person other than Dave as guilty
- Model states the problem is unsolvable or ambiguous
- Model skips constraint 2 and 4 interaction
- Model outputs only the answer without reasoning (partial failure)

---

## Success Conditions

- Model identifies Dave without error
- Model demonstrates constraint-by-constraint elimination
- Response is self-contained (no need for follow-up)
