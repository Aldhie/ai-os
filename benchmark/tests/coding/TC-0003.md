## Question

```
Implement an async Python function that:
1. Calls NVIDIA NIM API with stream=True
2. Correctly parses the SSE (Server-Sent Events) stream
3. Handles the [DONE] termination signal
4. Extracts and accumulates the thinking block (content between <think></think>)
5. Extracts and accumulates the final response text separately
6. Returns a structured result: {thinking: str, response: str, total_tokens: int}
7. Handles connection errors, timeout, and malformed JSON gracefully

API endpoint: https://integrate.api.nvidia.com/v1/chat/completions
Model: nvidia/nemotron-3-ultra-550b-a55b
```

---

## Expected Behaviour

- Uses `aiohttp` or `httpx` async client
- SSE parsing: split on `\n\n`, strip `data: ` prefix, skip `[DONE]`
- JSON parse each chunk: `chunk['choices'][0]['delta']['content']`
- State machine to detect `<think>` open/close tags across chunk boundaries
- Graceful error: returns partial result + error flag on connection drop
- `extra_body` injection for `/think` mode
