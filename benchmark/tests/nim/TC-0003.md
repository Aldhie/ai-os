## Question

```
NVIDIA Nemotron Ultra supports a `budget_tokens` parameter that controls
how many tokens the model can use for internal thinking before responding.

Context:
- Default budget: 10,000 tokens
- Maximum budget: 32,000 tokens  
- Minimum budget: 1,000 tokens
- Thinking tokens DO count toward total token usage
- Free tier: 1,000 requests/day (token limits apply)

For each task type below, recommend the optimal budget_tokens value and justify:
1. Simple factual question ("What is the capital of France?")
2. Complex mathematical proof
3. Multi-document synthesis (5 documents, produce executive summary)
4. Code debugging (find a bug in 200-line Python script)
5. Creative writing (short story, 500 words)
6. Strategic business analysis (SWOT for a given company)
```

---

## Expected Behaviour

| Task | Recommended Budget | Rationale |
|------|-------------------|----------|
| Simple factual | 1,000 (minimum) or disabled | No thinking needed; wastes tokens |
| Complex math proof | 16,000-32,000 | Deep step-by-step verification required |
| Multi-doc synthesis | 8,000-16,000 | Content integration requires planning |
| Code debugging | 10,000-16,000 | Hypothesis generation + code tracing |
| Creative writing | 2,000-5,000 | Light planning, but not deep reasoning |
| Business analysis | 8,000-12,000 | Structured framework application |

Model must justify each recommendation with reasoning about the task's cognitive demands.

---

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Recommends minimum/disabled for simple factual | 25% | Token conservation insight |
| Recommends maximum range for math proof | 25% | Understands depth requirement |
| Justified recommendations for all 6 tasks | 30% | Not arbitrary values |
| Considers token cost vs quality trade-off | 20% | Mentions free tier constraint |
