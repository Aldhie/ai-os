## Question

```
Session 1: User states: "I prefer long, detailed responses."
Session 2: User states: "Actually, I prefer concise responses."
Session 3: User asks a technical question.

Expected: AI uses the LATEST preference (concise).

Additional scenario:
Session 1: User states: "I like Python."
Session 4: User says: "I've been working on a Go project lately, use Go examples."
Session 5: User asks: "How do I reverse a string?"

Expected: AI uses Go (most recent context) but may acknowledge it can do Python if preferred.

Evaluate:
1. Does the AI correctly update contradictory preferences?
2. Does the AI use the most recent, contextually relevant preference?
3. Does the AI handle implicit preference changes (switch in context) correctly?
```

---

## Expected Behaviour

1. Session 3 response is concise (latest explicit preference wins)
2. Session 5 response uses Go (most recent explicit context)
3. For Go response: may optionally offer Python alternative but should not default to Python
4. AI does not "average" contradictory preferences (not "somewhat detailed")
5. AI does not re-ask about preferences when they were already stated

---

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Latest preference used (concise) | 35% | Session 2 overrides Session 1 |
| Go used in Session 5 | 35% | Most recent language context |
| No averaging of contradictions | 20% | Clean preference update |
| No re-prompting | 10% | Memory is functional |
